# Portfolio Project

This is a personal portfolio website built with React and Vite.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 16 or higher recommended)

## How to Run Locally

1.  **Open Terminal**: Open your terminal application.
2.  **Navigate to Project**:
    ```bash
    cd /Users/vyshakiyengar/Desktop/portfolio
    ```
3.  **Install Dependencies** (if you haven't already):
    ```bash
    npm install
    ```
4.  **Start Development Server**:
    ```bash
    npm run dev
    ```
5.  **View in Browser**: Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `src/components`: Reusable UI components (Hero, Navbar, etc.)
- `src/styles`: CSS files for styling
- `src/App.jsx`: Main application component

## Building for Production

To create a production build (for deployment):
```bash
npm run build
```
The output will be in the `dist` folder.
